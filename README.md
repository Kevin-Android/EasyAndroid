# EasyAndroid
持续性交付项目 主要汇总Android 一些适配方案

### 目录结构介绍
```
|-- app ------------------------------------ 项目app壳工程目录
|-- library -------------------------------- library不包含UI相关依赖库目录
|   |-- httplibrary ------------------------ http网络模块
|-- module --------------------------------- module包含UI相关依赖库目录
|   |-- audio_module ----------------------- 音频模块
|   |-- permission_module ------------------ Android权限模块
|   |-- video_module ----------------------- 视频模块
|-- other----------------------------------- 第三方相关依赖库目录
|   |-- favicon.ico ------------------------ 友盟统计模块
|-- poject_config--------------------------- 项目共用配置
|   |-- config.gradle----------------------- Android模块通用gradle配置
|-- .gitignore ----------------------------- Git仓库忽略规则配置文件
|-- README.md ------------------------------ 项目描述md文件
|-- build.gradle --------------------------- 项目build.gradle配置
|-- setting.gradle ------------------------- 项目gradle配置
```
